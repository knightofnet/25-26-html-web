Livre - Apprendre le CSS de A à Z

1999 €

Le CSS permet de styliser et de mettre en forme les pages web. Avec ce livre, vous apprendrez à maîtriser les bases du CSS, à créer des mises en page attrayantes et à rendre vos sites web plus professionnels. Que ce soit pour personnaliser votre blog, créer un portfolio en ligne ou développer des sites web interactifs, ce livre vous fournira les compétences nécessaires pour réussir. Ciblez vos balises HTML <div> et <span> pour une mise en forme précise et efficace. Etudiants de L1 Maths découvrez le CSS de manière ludique et efficace avec de vrais exercices de TD adaptés à votre niveau.

Découvrir les chapitres du livre :

  1. Les sélecteurs CSS : Cibler les éléments HTML
  2. La cascade et l'héritage : Comprendre la priorité des styles
  3. Le modèle de boîte CSS : Gérer les marges, les bordures et les paddings
  4. Les mises en page CSS : Flexbox et Grid
  5. Le responsive design : Adapter votre site à tous les écrans
